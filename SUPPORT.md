# Support

Use the redacted Bug Issue Form for reproducible non-security problems. Include Agent and Controller `vX.Y.Z`, OS/architecture, service manager, expected/actual behavior, and the smallest relevant log window. Never include runtime/enrollment tokens, complete install commands, Authorization headers, token/service files, databases, private keys, or notification credentials.

Security issues go through the private link in `SECURITY.md`. Compatibility and support periods are maintained in the Controller repository's `docs/COMPATIBILITY.md`. Support is best effort and starts from the latest listed stable combination.
